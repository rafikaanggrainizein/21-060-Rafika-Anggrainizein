HTML DAN CSS

Apa Itu HTML?

HTML (Hypertext Markup Language) adalah bahasa markah yang digunakan untuk membuat dan menyusun konten di web. Dengan HTML, kita bisa menambahkan teks, link, gambar, dan elemen lain ke dalam halaman web. HTML berfungsi sebagai kerangka dasar dari semua situs web, memungkinkan struktur konten yang semantik dan terorganisir.

Apa Itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain tampilan halaman web yang dibuat dengan HTML. CSS mengontrol layout, warna, font, dan gaya visual lain dari sebuah halaman, memungkinkan pengembang untuk membuat situs yang tidak hanya fungsional tetapi juga estetis menarik. CSS membantu memisahkan konten dari desain, sehingga memudahkan pengelolaan dan pembaruan situs.
